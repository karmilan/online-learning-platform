## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/karmilan/online-learning-platform.git
   ```
2. Install dependencies:
   ```bash
   cd backend / frontend
   npm install
   ```
3. Start the development server:
   ```bash
   cd backend / frontend
   npm run dev
   ```
